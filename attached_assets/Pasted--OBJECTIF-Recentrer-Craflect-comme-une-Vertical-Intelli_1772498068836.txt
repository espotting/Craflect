🟣 2️⃣ CONFIDENCE SCORE — V1 LOGIC
🎯 Objectif

Mesure la fiabilité statistique du signal de niche.
Pas une métrique de performance utilisateur.

2.1 Data Volume Score (40%)
volume_score = min(total_videos / 500, 1)
2.2 Pattern Consistency Score (40%)
consistency_score =
average(
dominant_hook_percentage,
dominant_structure_percentage,
dominant_format_percentage
) / 100
2.3 Signal Stability Score (20%)

Simplification V1 acceptable :

stability_score = 1 - normalized_variance_between_top_patterns

Si simplification nécessaire :

If top 3 dominant patterns all score > 70 → stability_score = 0.8–1
Else moderate value.

2.4 Final Formula
confidence_score =
(0.4 × volume_score) +
(0.4 × consistency_score) +
(0.2 × stability_score)

Affichage : 0–100%

🟣 3️⃣ SIGNAL STRENGTH (Analytics Layer)
🎯 Objectif

Mesurer la clarté actuelle du signal dans la niche.

Différent du confidence.

Confidence = fiabilité statistique
Signal Strength = dominance réelle des patterns

Formule V1
signal_strength =
average(
dominant_hook_percentage,
dominant_structure_percentage,
dominant_format_percentage
)

Affichage : 0–100%

Interprétation
Signal Strength	Meaning
< 40	Signal weak
40–60	Signal emerging
60–80	Strong signal
80+	Clear dominant patterns
