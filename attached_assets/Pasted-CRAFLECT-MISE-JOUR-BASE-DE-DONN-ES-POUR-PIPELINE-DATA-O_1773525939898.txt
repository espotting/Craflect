CRAFLECT – MISE À JOUR BASE DE DONNÉES POUR PIPELINE DATA

OBJECTIF

Préparer la base de données de Craflect pour supporter le pipeline d’ingestion vidéo, la classification Content DNA, la détection de patterns et la génération d’opportunités.

La taxonomie existante "Content DNA v1" reste la taxonomie principale utilisée par les workers (Hetzner).

La taxonomie Legacy reste utilisée uniquement pour le frontend.

Les modifications doivent être additives et ne doivent pas casser l’architecture existante.

---

TAXONOMIE OFFICIELLE

La taxonomie utilisée pour le Pattern Engine est Content DNA v1 :

Hook Types
Structure Types
Topic Clusters
Emotion Values
CTA Types
Visual Styles
Cut Frequency

Cette taxonomie est déjà implémentée et doit rester la base du système d’analyse.

---

NOUVEAU CHAMP À AJOUTER

Ajouter un champ supplémentaire dans la classification :

niche_cluster

Objectif

Regrouper les topics dans des macro-niches afin d’éviter le bruit lors de la détection de patterns.

Exemples

tech_business
creator_economy
productivity
marketing
lifestyle
