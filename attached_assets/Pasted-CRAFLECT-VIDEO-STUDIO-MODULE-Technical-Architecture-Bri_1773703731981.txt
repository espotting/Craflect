CRAFLECT — VIDEO STUDIO MODULE
Technical Architecture Brief

OBJECTIF

Ajouter un module Video Studio dans Craflect afin de transformer automatiquement une opportunité virale détectée par le moteur d’intelligence en vidéo prête à publier.

Workflow produit cible :

Opportunity → Script → Blueprint → Video → Export → Workspace

Le studio doit rester simple (type Canva) et template-driven.

Pas d’éditeur vidéo complexe.

---

ARCHITECTURE ACTUELLE

Pipeline actuel :

Scraping (Apify)
→ Ingestion Worker
→ Transcription Worker (Whisper)
→ Classification Worker
→ Scoring Worker
→ Pattern Engine
→ Opportunity Engine
→ Dashboard

Script Generator et Blueprint Generator existent déjà.

Le Video Studio doit être inséré après le Blueprint Generator.

Pipeline final :

Opportunity
→ Script Generator
→ Blueprint Generator
→ Video Studio
→ Workspace

---

PHASE 1 — VIDEO STUDIO CORE

Objectif : permettre aux utilisateurs de transformer un script en vidéo.

Modules :

video-studio service

Sous-modules :

storyboard builder
template engine
caption engine
video renderer
avatar provider (optionnel)

VIDEO RENDERING

Technologie recommandée : Remotion.

Pipeline :

Blueprint JSON
→ Template Engine
→ React Components
→ Remotion Renderer
→ MP4

VIDEO MODES

Le studio doit supporter plusieurs modes :

Avatar Video
photo upload → avatar talking head

Template Video
captions animées + b-roll + slides

Les avatars doivent rester optionnels.

TEMPLATES

Les templates doivent être générés à partir des patterns détectés.

hook_type
structure_type
niche_cluster

Template JSON exemple :

{
"template": "3_tips",
"platform": "tiktok",
"scenes":[
{"type":"hook","duration":2},
{"type":"tip","duration":4},
{"type":"tip","duration":4},
{"type":"tip","duration":4},
{"type":"cta","duration":2}
]
}

PHASE 2 — CLIP GENERATOR

Objectif : transformer une vidéo longue en shorts.

Pipeline :

upload video
→ Whisper transcript
→ segment detection
→ generate 3–5 clips

Sortie :

MP4 + thumbnail.

Les clips doivent être compatibles TikTok / Reels / Shorts.

---

PHASE 3 — CONTENT GAP ENGINE

Objectif : détecter automatiquement les opportunités de contenu sous-exploitées.

Principe :

patterns avec forte viralité
mais faible nombre de vidéos.

Pipeline :

Pattern Engine
→ Low Competition Detection
→ Opportunity Alert

Affichage dans le Dashboard :
